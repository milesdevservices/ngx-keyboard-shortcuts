How it works

In this demonstration we are creating a component level keybinding listener using just the service and no directive. The listener is created by using a constructor object and then calling the service's listen method. The listen method returns a handle for the listener allowing for removal in the ngOnDestroy Method.

This demo is currently set to listen for the key combination 'Ctrl + m'. Press the combination to trigger the alert and see it in action!
